# 🍺 PROJETO CERVEJINHA

![JAVA](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![MYSQL](https://img.shields.io/badge/MySQL-00000F?style=for-the-badge&logo=mysql&logoColor=white)
![SWING](https://img.shields.io/badge/UI-Swing-blue?style=for-the-badge)

O **CERVEJINHA** É UM SISTEMA DESKTOP DESENVOLVIDO PARA ENTUSIASTAS DE CERVEJAS QUE DESEJAM CATALOGAR SUAS DEGUSTAÇÕES, AVALIAR RÓTULOS E ACOMPANHAR ESTATÍSTICAS DE CONSUMO DE FORMA PERSONALIZADA.

---

## FUNCIONALIDADES

* **SISTEMA DE USUÁRIOS:** CADASTRO, LOGIN E EDIÇÃO DE PERFIL COM SEGURANÇA.
* **GESTÃO DE CERVEJAS:** CADASTRO DETALHADO (NOME, ESTILO, IBU, TEOR ALCOÓLICO, PAÍS E FABRICANTE).
* **GALERIA DE RÓTULOS:** VISUALIZAÇÃO GRÁFICA DAS FOTOS DAS CERVEJAS CADASTRADAS.
* **FILTROS AVANÇADOS:** BUSCA POR NOTA, ESTILO OU DATA DE DEGUSTAÇÃO.
* **ESTATÍSTICAS:** * MÉDIA DE NOTAS POR ESTILO.
    * RANKING DE CERVEJAS FAVORITAS.
    * CONTROLE DE DEGUSTAÇÕES POR MÊS.

## ESTRUTURA DO PROJETO (PADRÃO DAO)

O SOFTWARE SEGUE A ARQUITETURA **DAO (DATA ACCESS OBJECT)** PARA MELHOR ORGANIZAÇÃO E MANUTENÇÃO:

* **`MODEL`**: CLASSES QUE REPRESENTAM AS ENTIDADES (`CERVEJA.JAVA`, `USUARIO.JAVA`).
* **`DAO`**: CLASSES RESPONSÁVEIS PELA COMUNICAÇÃO COM O BANCO DE DADOS (`CERVEJADAO.JAVA`, `USUARIODAO.JAVA`).
* **`VIEW`**: INTERFACES GRÁFICAS EM SWING (`TELALOGIN.JAVA`, `TELACADASTROCERVEJA.JAVA`, ETC.).
* **`UTIL`**: GERENCIAMENTO DA CONEXÃO JDBC (`CONEXAO.JAVA`).

## CONFIGURAÇÃO DO BANCO DE DADOS

PARA RODAR O PROJETO, É NECESSÁRIO CRIAR AS TABELAS NO MYSQL. UTILIZE O SCRIPT ABAIXO:

```sql
CREATE DATABASE CERVEJINHA;
USE CERVEJINHA;

CREATE TABLE USUARIO (
    ID INT AUTO_INCREMENT PRIMARY KEY,
    NOME VARCHAR(100) NOT NULL,
    EMAIL VARCHAR(100) UNIQUE NOT NULL,
    SENHA VARCHAR(200) NOT NULL
);

CREATE TABLE CERVEJA (
    ID INT AUTO_INCREMENT PRIMARY KEY,
    NOME VARCHAR(100) NOT NULL,
    ESTILO VARCHAR(50),
    TEOR_ALCOOLICO DECIMAL(4,2),
    IBU INT,
    PAIS VARCHAR(50),
    FABRICANTE VARCHAR(100),
    DATA_DEGUSTACAO DATE,
    LOCAL_DEGUSTACAO VARCHAR(100),
    NOTA INT CHECK (NOTA >= 0 AND NOTA <= 10),
    COMENTARIOS TEXT,
    SUGESTAO TEXT,
    FOTO LONGBLOB,
    USUARIO_ID INT,
    FOREIGN KEY (USUARIO_ID) REFERENCES USUARIO(ID) ON DELETE CASCADE
);
```
